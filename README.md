# 🌦️ Weather App

A simple and responsive Weather App built with **React**, **Vite**, and **Tailwind CSS**. It fetches real-time weather data using the OpenWeatherMap API.

## ✨ Features

- 🔍 Search weather by city
- 🌡️ Current temperature
- ☁️ Weather condition
- 💧 Humidity
- 🌬️ Wind speed
- ☁️ Cloud percentage
- 🌍 City & Country
- ⚠️ Error handling
- ⏳ Loading state
- 📱 Responsive UI

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- OpenWeatherMap API

## 🚀 Installation

```bash
git clone https://github.com/yasirjunejo/weather-app.git
cd weather-app
npm install
npm run dev
```

- Functional Components
